解题报告-cf600c

发表于 2019-04-08 | 更新于 2019-04-30 | 分类于 ACM算法 | 评论数： | 阅读次数：

题目大意：

给你一个仅仅包含小写字母的字符串s.
你可以将某个字母换成另外一个字母,这被称为一次操作.
所有的操作完成之后,你可以任意调换其中字母的顺序(不计入操作).
你需要用最小的步骤数将所给字符串变成回文串,如果有多种步骤数最小的情况,那么请你输出字典序最小的回文串.
Input
输入只有一个,包含一个长度小于等于2*10^5的非空字符串(只包含小写字母).
Output
请你输出符合要求的回文串。
Examples
【样例输入】
aabcd

【样例输出】
abcba

错误总结：

整体思路比较好像，将字符出现的次数存起来，然后从最小的开始便利，出现不是偶数个的字母，从后面最大的字母开始，不是偶数个的字母处转换过来一个
主要是有一些细节处理，如知道了 剩下哪些字母，每个多少，如何生成回文串，，巨坑，详情 看代码
