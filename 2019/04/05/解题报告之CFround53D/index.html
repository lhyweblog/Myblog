解题报告之CFround53D

发表于 2019-04-05 | 分类于 ACM算法 | 评论数： | 阅读次数：

题目大意：

有一个环形的摊位,总共有n个摊位.
每个摊位的糖果价格为a[i].
(n号摊位和1号摊位相邻).
现在小x从1号摊位开始,进行如下的买糖果策略:
1.如果他有足够的剩余的钱能够在第i号摊位买一个糖果,那么它就在该摊位买一个糖果.
2.决策完之后,它走到下一个摊位i+1号摊位,如果i=n的话,那么下一个摊位被定义为1号摊位.
问你如果初始的时候有T单位的钱的话,能买多少个糖果.

输入：
第一行两个整数n,T(1<=n<=2*10^5,1<=T<=10^18).
第二行有n个整数a1,a2…an(1<=ai<=10^9).

输出：
输出一个整数,表示小x最后买下的糖果的个数.

解题思路：

模拟买糖过程，遍历一圈，找出一圈必须要买得钱数sum和个数cnt，然后用剩余总钱数去算(T/sum*cnt)，这种情况下能买到什么时候以及买多少
再跟新 剩余钱数T = T%sum。然后接着遍历 看剩下T钱得情况下需要怎么买，知道T小于最小得糖果钱数，模拟结束，跳出模拟。
